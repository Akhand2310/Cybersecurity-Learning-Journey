# Inside a Computer System

## Topics Learned

- Power Supply Unit (PSU)
- Firmware (UEFI/BIOS)
- POST (Power-On Self Test)
- Boot Device Selection
- Bootloader
- Operating System Startup

## Key Takeaways

1. The PSU provides power to all components.
2. Firmware (UEFI/BIOS) initializes hardware.
3. POST checks if hardware is functioning correctly.
4. The firmware locates a bootable device.
5. The bootloader loads the operating system into RAM.
6. The operating system takes control of the system.
